# Bakery E-Commerce Web Application

## Overview
This is a fully functional Bakery E-Commerce Web Application built using HTML5, CSS3, Bootstrap 5.3, Bootstrap Icons, and Vanilla JavaScript. The application allows users to browse products, manage their shopping cart, and complete the checkout process seamlessly.

## Features
- **Landing Page**: The main entry point of the application featuring a sticky navbar, hero section, category cards, featured products, testimonials, and footer.
- **Product Listing**: A dedicated page to display products with live search, pagination, and CSV export functionality.
- **Shopping Cart**: A page that shows the user's selected items, order summary, coupon application, and options to continue shopping or proceed to checkout.
- **Checkout Process**: A streamlined checkout page that collects delivery information, payment methods, and displays a progress bar.
- **Order Confirmation**: A page that confirms the order with an order ID, estimated delivery date, and order summary.

## Setup Instructions
1. Clone the repository to your local machine.
2. Open the `index.html` file in your web browser to view the application.
3. Ensure you have a local server running if you want to test features that require JavaScript fetching data.
