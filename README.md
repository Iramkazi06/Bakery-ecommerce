# 🧁 Bakery E-Commerce(Sweet Bite)

A full-featured online bakery store built with HTML, CSS, and JavaScript.

## 🍞 Features

- 🏪 **Shop** – Browse all available bakery products
- 🛒 **Cart & Checkout** – Add to cart and place orders seamlessly
- 👤 **Customer Dashboard** – Track orders and manage your account
- 🔐 **Login & Register** – User authentication pages
- 🧑‍🍳 **Vendor Dashboard** – Manage products and orders as a vendor
- 📦 **Order Confirmation** – Post-purchase confirmation page

## 🛠️ Tech Stack

- **HTML5** – Page structure
- **CSS3** – Styling (in `/css` folder)
- **JavaScript** – Interactivity (in `/js` folder)
